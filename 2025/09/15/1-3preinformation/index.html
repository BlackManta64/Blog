TrayPerf 1.3pre Information

Posted on 2025-09-15

Update information

  • Support for circular graphs

Problem / Bug

Label Update During Language Switch

  • Labels may display 0% immediately after switching.
    Graph Type Switch Reflection Delay
  • Icons may not update instantly after changing graph_type.
    GPU Usage Retrieval Exception Risk
  • If GPUtil.getGPUs() throws an exception, update() continuously outputs errors.
    Pie Chart Drawing Decimal/0% Issue
  • Pie segments may not render when percent values are small.
    Network Speed Display Limitations
  • Bars/pie charts may not accurately reflect speeds exceeding 100 kB/s.
    Use of global / nonlocal
  • State management is scattered, potentially causing conflicts during extensions.
    Duplicate Menu Construction
  • Creating identical menus in each tray increases memory consumption.
    Fixed Translation Keys
  • Adding new labels may cause translations to be missed.
